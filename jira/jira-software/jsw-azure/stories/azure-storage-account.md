\# Story: Configure Azure Storage Account

* Create Azure Storage Account
* Select appropriate performance tier (Standard)
* Configure redundancyoption (LRS/GRS/ZRS/Ra-GRS/Ra-ZRS  as required)
* Use Storage account for:

&nbsp;  - Boot Diagnostics

&nbsp;  - VM Logs and Analytics

&nbsp;  - Backup and monitoring dependancies

* Restrict public network access if not required
* Enable secure transfer (HTTPS)
* Validate storage account connectivity and usage
