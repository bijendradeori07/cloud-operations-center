\# Epic: Azure Security, Monitoring \& Backup



\## Epic Objective:

Implement security controls, monitoring, alerting \& backup strategies to ensure availability, observability and recoverability of Azure workloads.



\## Scope

* Network security enforcement
* Centralized monitoring and logging
* Alerting for availability and performance
* Backup and recover configuration
